# Cron-Backup Manager

Bu proje, Açık Kaynak İşletim Sistemleri dersi kapsamında geliştirilmiş bir terminal otomasyon aracıdır.

## Özellikler
- **Zamanlanmış Görev Yönetimi:** Linux Crontab altyapısı ile tam uyum.
- **Otomatik Yedekleme:** Python subprocess modülü ile güvenli dosya sıkıştırma.
- **Hata Yönetimi:** Unix I/O standartlarına (stdout/stderr) uygun loglama.

## Proje Yapısı
- `src/`: Ana kaynak kodlar (Python).
- `specs/`: Proje teknik gereksinimleri ve JSON metadata bilgisi.
- `researchs/`: Yapay zeka modelleri ile yapılan teknik araştırmalar.
- `docs/`: Kullanım kılavuzu ve belgeler.
